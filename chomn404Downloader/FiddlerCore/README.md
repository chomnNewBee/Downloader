这是FiddlerCore 5.0.2版本源码，支持.net framework和.net core,百分百c#源码，功能全而且免费，无功能限制，还修复了多处bug。

介绍：
FiddlerCore可捕获和修改 HTTP 和 HTTPS 流量，而无需任何 Fiddler UI。

 特点：
- HTTP 和 HTTPS 流量捕获和修改。
- 用于内容过滤和修改的强大对象模型。
- 存储和重新加载网络流量。
- 支持几乎任何客户端应用程序。
- 通过移动代理设置支持大多数设备。

适合人群：
具备一定编程基础，1年以上开发经验的人使用

使用方式：
直接添加到你的项目即可
